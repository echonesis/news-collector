🔌 Extension 安裝指南

1. 開啟 Chrome 瀏覽器，進入 chrome://extensions/
2. 啟用右上角的「開發人員模式」
3. 點擊「載入未封裝項目」
4. 選擇 news-collector/extension 資料夾
5. Extension 應該會出現在工具列中

✅ Extension 功能測試

1. 點擊工具列上的 News Collector 圖示
2. 填寫訂閱表單：
  • 主題: AI 人工智慧
  • 頻率: daily
  • Email: 您的郵箱地址
3. 點擊「訂閱」按鈕
4. 如果成功，應該會顯示「訂閱成功！」訊息

🛠️ 故障排除

如果 Extension 無法連接：
  • 檢查 Chrome 開發者工具 (F12) 的 Console 是否有錯誤
  • 確認網路連線正常
  • 檢查 CORS 設定
